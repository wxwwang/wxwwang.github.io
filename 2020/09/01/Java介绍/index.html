Java语言概述

Java语言诞生于1995年，Java是sun公司开发的一门编程语言，目前被Oracle公司收购，编程语言就是用来编写软件的。

Java的应用

​ *开发QQ、微信程序（桌面应用软件）

​ *淘宝、京东（互联网应用软件）

​ *安卓应用程序

Java语言平台

​ *Javase标准版，Javase并不能开发大型项目。

​ *Javaee企业版，学习完Javaee部分就可以开发各种大型项目了。

Java语言开发环境

jdk是Java开发环境，官网http://www.oracle.com/cn/index.html

jdk的安装

傻瓜式安装，双击安装程序，然后一路next，安装的方式：

​ *安装路径不要有中文和特殊符号如空格等。

​ *所有和开发相关的软件做好安装目录统一。

验证安装是否成功，通过dos命令，切换到jdk安装的bin目录下。比如D:\java\jdk\bin，然后输入javac和Java，如果正常显示一些内容，说明安装成功。

配置环境

环境变量的作用：由于javac和java命令只能在固定的目录下才能执行，而我们写的代码如果都和javac及Java命令放在相同的目录的话，会显得很凌乱，为了让java的bin目录下的java命令可以在任意目录下执行，就得配置环境变量。

系统变量—>新建变量名JAVA_HOME–>变量值D:\java\jdk

系统变量–>新建变量名CLASSPATH–>变量值**.;%JAVA_HOME%\lib;%JAVA_HOME%\lib\tool.jar**

环境变量双击进入path–>新建**%JAVA_HOME%\bin**–>新建**%CLASSPATH%**

之后一直点确认就ok了。

然后打开cmd，任意位置输入javac，下图这样就成功了。

jdk和jre以及跨平台

jdk与jre的关系

​ *jdk：Java语言的开发工具包，提供了Java语言的开发工具，它里面包含了jre，同时也包含了JVM(java虚拟机)。

​ *jre:是Java运行环境，如果你不需要开发只需要运行Java程序，那么你就可以安装jre。

跨平台特性：平台指的是操作系统。只需要在相应平台上安装Java虚拟机，就可以运行java程序。

Java的加载与执行

编译阶段

编译阶段主要的任务是检查Java源程序是否符合Java语法，符合就生成字节码文件，不符合就无法生成字节码文件.class文件。

程序员只需在硬盘的某个位置新建一个.java扩展名的文件，该文件成为源文件，源文件当中编写的是Java源代码。

而整个源代码不是随意编写的，必须符合Java语法规则https://files-cdn.cnblogs.com/files/han-1034683568/阿里巴巴Java开发手册终极版v1.3.0.pdf。

Java程序员需要使用jdk当中自带的javac.exe命令进行Java程序的编译。

javac的使用规则

​ javac Java源文件的路径

运行阶段

Java程序员需要使用jdk当中自带的java.exe命令进行Java程序的运行。

​ Java 类名

运行过程

java.exe命令会启动Java虚拟机jvm，jvm会启动类加载器ClassLoader,ClassLoader会去硬盘上去搜索字节码文件，将该文件加载到jvm中，jvm将字节码文件解释成二进制这样的数据，然后操作系统执行二进制和底层硬件平台进行交互。
