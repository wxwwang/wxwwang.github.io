8.快捷键

常用快捷键链接https://blog.csdn.net/qq_30764991/article/details/88082743

1.psvm + 回车：main方法

2.sout + 回车：输出语句

3.ctrl + d:复制当前语句

4.alt + alt +L:格式化代码 和qq中的热键冲突，需要修改qq的快捷键

5.alt + shift + 向上的箭头 ：向上移动代码

6.alt + shift + 向下的箭头 ：向下移动代码

7.ctrl + shift + /:多行注释

8.ctrl + /:单行注释

9.alt + 回车：代码修正

10.alt + /:自动提示

11.ctrl + y :删除当前行

12.shift + 回车：向下换行

13.ctrl + alt +回车：向上换行

14.ctrl + shitr + 回车：自动补全代码

15.alt + insert：创建包/类/模块的快捷键

​ *有的笔记本电脑insert键位的使用需要按住fn键

​ *有的笔记本电脑的insert键被简写成ins

16.修改类/包的名称

​ *在类名的位置右键–>Refactor–>Rename

​ *快捷键：shift +F6

​ *直接修改class关键字后面的名称，alt + 回车 –>Rename file

17.ctrl + F2 :终止程序运行

18.10.fori + 回车：for(int i = 0;i < 10;i++){}
