480个Linux指令

Linux中没有错误就代表操作成功

du指令

du：检查磁盘空间使用量

只列出当前目录下的所有文件夹容量（包括隐藏文件夹）

col

过滤掉控制字符

cat mytest | col

vi

编辑文件或者创建文件并编辑

colrm

显示指定列的字符

colrm 20 < /home/1.txt //显示前20列的字符

comm

两个文件之间的比较

使用前提，文件必须使用sort进行排序

csplit

分割文件

csplit 1.txt 300 //将文件分割成两个文件，以300行为分割点

ed

以行编辑文件

egrep

查找指定字符串

egrep -i ‘hello’ 文件路径 //-i忽略大小写

ex

ex是vi的单行编辑指令

fgrep

查找指定的字符串

fmt

按照指定格式显示文件内容

fold

指定显示文件的宽度

fold -w 20 文件名 //显示宽度为20个字符

grep

显示包含指定字符的行

grep root 文件名

ispell

英语拼写检查

join

将两个文件重复的字段只显示一个

look

查找以指定字符开头的文件 文件必须使用sort排序后使用

look 单词 //查询以单词开头的全部单词，相当于字典

mtype

类似于cat

rgrep = grep -R

递归查找文件内容

sed

类似cat ,但是可是指定不显示哪一行，不改变源文件

sort

排序输出，不改变原文内容

spell

检查文件中错误的单词，并输出让我们检查

tr

字符处理工具，替换文件中的字符

uniq

过滤掉重复的行

wc

diff

用于比较两个文件不同的地方

diffstat

用于对diff输出的文件进行统计

file

file 文件

看文件是什么类型

find

查找文件

find / -name passwd

find /root -name mylinux -exec rm -f {} ;
