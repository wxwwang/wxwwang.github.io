480个Linux指令

Linux中没有错误就代表操作成功

du指令

du：检查磁盘空间使用量

只列出当前目录下的所有文件夹容量（包括隐藏文件夹）

col

过滤掉控制字符

cat mytest | col

vi

编辑文件或者创建文件并编辑

colrm

显示指定列的字符

colrm 20 < /home/1.txt //显示前20列的字符

comm

两个文件之间的比较

使用前提，文件必须使用sort进行排序

csplit

分割文件

csplit 1.txt 300 //将文件分割成两个文件，以300行为分割点

ed

以行编辑文件

egrep

查找指定字符串

egrep -i ‘hello’ 文件路径 //-i忽略大小写

ex

ex是vi的单行编辑指令

fgrep

查找指定的字符串

fmt

按照指定格式显示文件内容

fold

指定显示文件的宽度

fold -w 20 文件名 //显示宽度为20个字符

grep

显示包含指定字符的行

grep root 文件名

ispell

英语拼写检查

join

将两个文件重复的字段只显示一个

look

查找以指定字符开头的文件 文件必须使用sort排序后使用

look 单词 //查询以单词开头的全部单词，相当于字典

mtype

类似于cat

rgrep = grep -R

递归查找文件内容

sed

类似cat ,但是可是指定不显示哪一行，不改变源文件

sort

排序输出，不改变原文内容

spell

检查文件中错误的单词，并输出让我们检查

tr

字符处理工具，替换文件中的字符

uniq

过滤掉重复的行

wc

diff

用于比较两个文件不同的地方

diffstat

用于对diff输出的文件进行统计

file

file 文件

看文件是什么类型

find

查找文件

find / -name passwd

find /root -name mylinux -exec rm -f {} ;

gitview -c

类似于vi，但是可以彩色显示

ln

创建软硬连接

ln 文件名 新文件名 硬连接

ln -s 文件名 新文件名 软连接

locate

从数据库中查找数据，速度快，要经常执行update，以保证查询到的数据的准确性

lsattr

显示指定文件的dir属性

mcopy

复制文件

mdel

删除文件

rmdir

用于删除空目录

mmove

移动文件或者重命名

mread复制

mren移动

mshowfat

获取dos文件信息

mtools

激活dos操作系统中的指令

mv 移动和重名名

rev 以行反序输出文本

paste

多个文件以列合并

patch

为源代码打安全补丁

rcp

两台主机之间，无密码复制文件

rhmask

加密文件

rhmask 要加密的文件 保存加密的文件

rm

删除文件或者文件夹

slocate

查找指定字符或者字符串的文件名，或者文件内容

slocate 字符串 回车

split

将大文件分割成若干小文件

tee

将指定内容保存到多个文件

tmpwatch

删除临时文件

touch

创建文件

umsdk

看文件的执行权限

which

查看要查询指令的全称和路径

which ls

cat

显示文件内容

whereis

显示要查询指令的位置

whereis ls

chattr

可以给文件加权限，+i 后root用户也不能删除，想要删除，就chatter -i 文件名 。之后在删除。

chgrp

改变文件的工作组，chgrp 要保存的组 文件名

chmod

改变文件的权限

chown

该文件的所属主，和工作组。

chown 主：组 文件名

cksum

查看文件校验和，看看传输后有没有改变内容。

cmp

用于比较两个文件的不同。

cp

复制文件或目录，复制目录要加 -R。

cut

查看指定列的内容

indent

改变文件格式

ftp

进入ftp模式

bye

推出ftp模式

ftpcount

显示ftp当前连接用户数

ftpshut

在指定时间停止ftp服务

ftpwho

显示当前ftp会话

ncftp

增强版的ftp

fftp

上传和下载文件

uucico

获取远程服务器的文件列表

uucp

两个系统之间的文件传输

uupick

对主机传送过来的数据进行处理

uuto

将本地文件传输到远端主机

ar

压缩文件或者解压文件

gzip

压缩指令

gunzip

解压gzip压缩的文件

bzip2

压缩文件

bunzip2

解压文件

cpio

备份文件

dump

备份文件系统

restore

还原dump创建的备份文件

compress

压缩文件

uncompress

解压文件

zip

压缩文件

unzip

解压文件

arj

压缩问价

unarj

解压文件

bzcat

显示压缩文件的内容

bzcmp

比较两个压缩文件的不同

ss

当前套接字的信息

iptraf

实时的监视网卡的流量

mysqldump

备份数据库

mysqladmin

创建新的数据库

mysqlimport

向指定的数据库或表导入指定的数据

mysqlshow

显示数据库所有表，和表的类型

mysql

连接数据库

nfsstat

显示客户端的信息

showmount

显示nfs服务的加载信息

smbpasswd

设置密码或者重新设置密码

sendmail

sendmail的启动

smbclient

连接远程服务器

reboot

重启系统

renice

调整正在运行线程的优先级

rlogin

远程登录

rsh

在本地控制远程服务器进行指定操作

rwho

获取系统用户

shutdown

关机

su

切换用户

suspend

暂停指令

swathch

调用系统监控程序。

uname

打印当前系统内核的版本号

usersdd

添加用户

userdel

删除用户

usermod

改变用户信息

vlock

让当前程序或者窗口进入睡眠状态

w

groupadd

aumix

apmd

bind

declare

设置变量

depmod

内核映射关系

dircolors

进行颜色的配置

显示系统启动时的内核信息

enable

用于激活和关闭内部指令

eval

用于shell脚本编程，eval ls = ls
